Detailed Description

template<typename T = int>
class CArray2D< T >

szablon klasy pochodnej szablonu wektora z biblioteki STL realizującej wektor dwuwymiarowy o ustalonych przy tworzeniu wymiarach

umożliwia dostęp do elementów zarówno przez odwołanie się po dwóch współrzędnych jak i po jednej współrzędnej (przy odczycie wierszami poczynając od elementu (0,0)

Ttyp przechowywany w wektorze, domyślnie int
