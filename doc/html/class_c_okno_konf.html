konstruktor obiektu okna konfiguracyjnego

konstruktor inicjuje obiekt ui oraz ustawia tekst na etykietach opisujących spinboxy z liczbą kolumn i liczbą wierszy (podając możliwą do ustawienia wartość minimalną i maksymalną) i parametryzuje kontrolki spinbox ustawiając ich wartości minimalne i maksymalne oraz aktualne wartości wszystkie dane odczytyuwane są z mapy przekazanej jako parametr za pomocą wyszukiwania odpowiednich kluczy jesli klucz nie zostanie znaleziony - nadawana jest wartość domyślna

Parameters
[in]parammapa parametrów - zarówno klucz jak i wartość są zmiennymi łańcuchowymi
[in]kontroler- wskaźnik na obiekt kontrolera - do ewentualnego przesyłania informacji o akcji uzytkownika za pomocą metody CKontroler::wyslijWiadomosc
