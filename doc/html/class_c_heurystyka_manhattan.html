Detailed Description

klasa realizująca heurystykę opartą na sumie odległości (w metryce miejskiej) każdego elementu układanki od jego domyślnej pozycji

wartośc tej funkcji heurystyki jest równa sumie odległości (w metryce miejskiej) każdego elementu układanki od jego domyślnej pozycji (nie uwzględniając połozenia pola pustego)
