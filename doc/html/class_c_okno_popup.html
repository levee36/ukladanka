konstruktor obiektu okna popup z krótką wiadomością

konstruktor inicjuje obiekt ui oraz ustawia:

  • tytuł okna
  • treść wyświetlanej na etykiecie wiadomości
  • tekst na przycisku
    powyższe dane pobierane są z przekazanego jako parametr słownika-mapy przez szukanie odpowiednich kluczy (tytul,tresc,przycisk) jeśli któryś z kluczy nie zostanie odnaleziony w mapie- nadawana jest wartość domyślna
